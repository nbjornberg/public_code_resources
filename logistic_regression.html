Logistic Regression

Logistic regresison is for preditcing a discrete outcome, such as yes or no. The analysis assumptions are not strict: does not assume normality, linearity, or equal group variance, and multiple types of variables can be used as predictors. - The output of this analysis is odds.

Assumption Testing

  1. Linear relationship between continuous predictors and logit transform of DV (Box-Tidwell)
  2. Multicollinearity
  3. Outliers (standardized residuals)
  4. Independent responses
# SET VARIABLES BEFORE RUNNING

### 1. Test for linear relationships
# Fit the logistic regression model
model <- glm(promoted ~., 
             data = sp_clean, 
             family = "binomial")
# Predict the probability
probabilities <- predict(model, type = "response")
predicted.classes <- ifelse(probabilities > 0.5, "pos", "neg")

# select predictors
df <- sp_clean %>% 
  select_if(is.numeric)   
  
predictors <- colnames(df)

# calculate logits based on probabilities
df <- df %>%
  mutate(logit = log(probabilities/(1-probabilities))) %>%
  gather(key = "predictors", value = "predictor.value", -logit)

# visualize; review for linearity and determine if transformations are needed; ignore dummy coded variables
df %>% 
  ggplot(aes(logit, predictor.value))+
  geom_point(size = 0.5, alpha = 0.5) +
  geom_smooth(method = "loess") + 
  facet_wrap(~predictors, scales = "free_y") +
  labs(title = "Test for linearity of variables and logit of outcome")
